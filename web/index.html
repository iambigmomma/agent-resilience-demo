唯一的差別

兩條 lane 跑的是同一份 agent.py，打的是同一個故障。 client.py 裡沒有任何一行 if lane ==。差別只有 lanes.py 裡一個 tuple 的長度 —— 右邊那條有地方可以去。

# lanes.py SINGLE = Lane(candidates=(PRIMARY,)) # ← 無路可退，燒完 retry 就死 ROUTED = Lane(candidates=(PRIMARY, ALT)) # ← 繞過故障，完成任務

這不是延遲測試。routed lane 的 wall clock 有時更慢（多打了一次失敗的請求）。 重點只有一個：任務有沒有完成。故障是本機注入的合成 429，不代表任何供應商的真實限制。
